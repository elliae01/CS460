Directions to run application.

1. Set-up a raspberry pi with the newest version of raspbian.
2. Install Andaconda (https://anaconda.org/anaconda/python).
3. Follow set-up isntructions supplied with Andaconda.
4. Copy the files in this folder to a single location on the raspberry pi. 
5. Connect all devices including (Piezo and Servo motor). (diagram included to show how connection are to be done 'target')
6. Navigate to the location where the files have been uploaded.
7. type 'Python3 Target.py'.
