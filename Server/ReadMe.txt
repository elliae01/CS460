Directions to run application.

1. Install Andaconda (https://anaconda.org/anaconda/python).
2. Follow set-up isntructions supplied with Andaconda.
3. Copy the files in this folder to a single location on the pc.
4. Install Oracle Database 11g Enterprise.
5. Follow directions on oracle setup.
6. Run the Server.py by clicking on it at the same time you are running Client and Target scripts on 2 seperate raspberry pis.
7. Wait for the application to be completed.
8. Run the GUI.py by clicking on it.
